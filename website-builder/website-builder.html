Onboarding

Following the guided instructions to add restaurant name and description, logo and favicon image files, and to choose some of the initial design options for the website.

📝
Rich Text Editing

To edit content on a page, select the element that you want to make changes, type to make changes directly on the page.

♿
Third-Party Accessibility Tool (UserWay)

Enter UserWay, an accessibility widget that puts the power to make those adjustments into the guest’s hands. Restaurants now have the option of adding a widget to their site that opens a range of accessibility options to their patrons.

🦾

Pixel Tracking (Meta or Google)

Marketing and analytics settings allows users to set up pixel tracking IDs from Facebook/Meta and Google Analytics. These allow them to track traffic from these sites to the website or Toast Online Ordering Pro site.

🔍
SEO

All the websites that users create with Toast are search engine optimized (SEO), making it easy for guests to find the restaurant first.

🖥
Domain

Domain is the web address for the restaurant (or brand) page, connecting a domain will allow user to publish the website and all the related pages they’ve created. Users can purchase a domain directly from the configuration page in Toast Web.
